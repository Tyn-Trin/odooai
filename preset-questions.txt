================================================================================
 คำถามที่ระบบตอบได้โดยไม่เสีย token (Preset + Classifier)
 อัปเดต: 2026-06-24
================================================================================

ระบบมี 3 tier:
  Tier 1  Keyword match  →  0 token 0.005 บาท
  Tier 2  AI Classify    →  ~10 token (แค่จำแนกประเภท) 0.04 บาท
  Tier 3  Full Claude    →  500-3,000+ token (คำถามเฉพาะเจาะจง) 0.24-1.50 บาท

--------------------------------------------------------------------------------
 1. CEO_OVERVIEW  —  ภาพรวมธุรกิจ
--------------------------------------------------------------------------------
Tier 1 (keyword): วันนี้, เป็นยังไง, ภาพรวม, สรุป, overview, summary
Tier 2 (AI):      ธุรกิจดีไหม / มีอะไรน่าเป็นห่วงไหม / สถานการณ์ตอนนี้เป็นอย่างไร
                  เดือนนี้โอเคไหม / อัปเดตหน่อย / ภาพรวมเดือนนี้

ข้อมูลที่ได้:  ลูกหนี้ / เจ้าหนี้ / ยอดขาย / SO ค้างส่ง / PO รอวาง bill

--------------------------------------------------------------------------------
 2. OVERDUE_AR  —  ลูกหนี้เกินกำหนดชำระ (Aging)
--------------------------------------------------------------------------------
Tier 1 (keyword): เกินกำหนด, เลยกำหนด, เกิน 30, เกิน 60, เกิน 90, aging
Tier 2 (AI):      ใครค้างนานสุด / ลูกหนี้เก่า / หนี้เสีย / ค้างมานานแล้ว
                  ใครไม่จ่ายมาหลายเดือนแล้ว

ข้อมูลที่ได้:  จำนวนและยอดแยก bucket: 1-30 / 31-60 / 61-90 / >90 วัน

--------------------------------------------------------------------------------
 3. RECEIVABLE  —  ลูกหนี้ทั้งหมด
--------------------------------------------------------------------------------
Tier 1 (keyword): ลูกหนี้, receivable, ค้างชำระ, ค้างรับ
Tier 2 (AI):      ใครยังไม่จ่ายเงิน / เงินค้างรับมีเท่าไร / AR ทั้งหมด

ข้อมูลที่ได้:  ยอดรวม / เกินกำหนดแล้ว / ตัวอย่าง 20 รายการ

--------------------------------------------------------------------------------
 4. MONTHLY_SALES  —  ยอดขายเดือนนี้
--------------------------------------------------------------------------------
Tier 1 (keyword): ยอดขาย, รายได้, ขายได้
Tier 2 (AI):      ขายได้เท่าไรเดือนนี้ / รายได้เดือนนี้เป็นยังไง / sales เดือนนี้

ข้อมูลที่ได้:  ยอดรวม invoice / จำนวน invoice / ตัวอย่าง 20 รายการ

--------------------------------------------------------------------------------
 5. PO_PENDING_BILL  —  PO รอวาง bill
--------------------------------------------------------------------------------
Tier 1 (keyword): po ค้าง, รอวาง bill, รอบิล, ค้าง bill
Tier 2 (AI):      รับของแล้วแต่ยังไม่ได้ bill / PO ที่ยังไม่มี invoice

ข้อมูลที่ได้:  จำนวน PO / ยอดรวม / ตัวอย่าง 20 รายการ

--------------------------------------------------------------------------------
 6. AP_SUMMARY  —  เจ้าหนี้รอจ่าย
--------------------------------------------------------------------------------
Tier 1 (keyword): เจ้าหนี้, payable, รอจ่าย, ต้องจ่าย
Tier 2 (AI):      ต้องจ่ายเงินใครบ้าง / bills ที่ยังค้างอยู่ / ค่าใช้จ่ายค้างจ่าย
                  มีบิลไหนใกล้ครบกำหนดไหม / ควรจ่ายเงินใครก่อน

ข้อมูลที่ได้:  ยอดรวมทั้งหมด / เกินกำหนดแล้ว / ครบกำหนดใน 7 วัน / ตัวอย่าง 20

--------------------------------------------------------------------------------
 7. SO_PENDING_DELIVERY  —  SO ค้างส่งสินค้า
--------------------------------------------------------------------------------
Tier 1 (keyword): so ค้าง, ค้างส่ง, ยังไม่ส่ง, ค้างจัดส่ง
Tier 2 (AI):      ออเดอร์ไหนยังไม่ส่ง / ลูกค้ารอของอยู่ / delivery ค้าง
                  ส่งของไม่ทันไหม / มี SO ที่ยังไม่จัดส่งไหม

ข้อมูลที่ได้:  จำนวน SO / ยอดรวม / ตัวอย่าง 20 รายการพร้อม commitment date

--------------------------------------------------------------------------------
 8. TOP_CUSTOMERS  —  ลูกค้า top 10 เดือนนี้
--------------------------------------------------------------------------------
Tier 1 (keyword): top customer, ลูกค้า top, ลูกค้าอันดับ, ลูกค้าซื้อมาก
Tier 2 (AI):      ใครซื้อมากสุดเดือนนี้ / ลูกค้าหลักเราคือใคร / ลูกค้าสำคัญ
                  ลูกค้า VIP / ใครสร้างรายได้มากสุด

ข้อมูลที่ได้:  ลูกค้าอันดับ 1-10 พร้อมยอดซื้อเดือนนี้

--------------------------------------------------------------------------------
 9. STOCK_LOW  —  สินค้าสต็อกหมด / ติดลบ
--------------------------------------------------------------------------------
Tier 1 (keyword): สต็อกหมด, ของหมด, สินค้าหมด, ขาดสต็อก, stock ต่ำ
Tier 2 (AI):      สินค้าไหนหมดคลัง / ของขาดมีอะไรบ้าง / สต็อกติดลบ
                  สินค้าที่ต้องสั่งด่วน / ของในคลังเหลือน้อย

ข้อมูลที่ได้:  รายการสินค้าที่ qty_available <= 0 (สูงสุด 50 รายการ)

--------------------------------------------------------------------------------
 10. PO_PENDING_RECEIPT  —  PO รอรับสินค้า
--------------------------------------------------------------------------------
Tier 1 (keyword): รอรับสินค้า, ยังไม่รับ, ของยังไม่มา
Tier 2 (AI):      สั่งของไปแล้วแต่ยังไม่ได้รับ / vendor ยังไม่ส่งของ
                  PO ที่รอของอยู่ / ของที่ยังไม่เข้าคลัง

ข้อมูลที่ได้:  จำนวน PO / ยอดรวม / ตัวอย่าง 20 รายการพร้อม date_planned

--------------------------------------------------------------------------------
 11. PAYMENT_RECEIVED  —  รับเงินจากลูกค้าเดือนนี้
--------------------------------------------------------------------------------
Tier 1 (keyword): เงินเข้า, รับชำระ, ลูกค้าจ่าย
Tier 2 (AI):      เดือนนี้ได้รับเงินจากลูกค้าเท่าไร / cash inflow เดือนนี้
                  ลูกค้าจ่ายเงินมาแล้วเท่าไร / เงินเข้าเดือนนี้มีเท่าไร

ข้อมูลที่ได้:  จำนวนรายการ / ยอดรวม / ตัวอย่าง 20 รายการ

--------------------------------------------------------------------------------
 12. EXPENSE_SUMMARY  —  ค่าใช้จ่าย / บิลซื้อเดือนนี้
--------------------------------------------------------------------------------
Tier 1 (keyword): ค่าใช้จ่าย, บิลซื้อ, expense
Tier 2 (AI):      เดือนนี้ซื้อของไปเท่าไร / ใช้เงินไปเท่าไรเดือนนี้
                  ค่าใช้จ่ายรวมเดือนนี้ / vendor เรียกเก็บเงินมาเท่าไร

ข้อมูลที่ได้:  จำนวน bill / ยอดรวม / ตัวอย่าง 20 รายการ

================================================================================
 คำถามที่ยังต้องใช้ Full Claude (Tier 3)
================================================================================

- คำถามที่ระบุชื่อ partner เฉพาะ
    เช่น "บริษัท ABC ค้างชำระเท่าไร"

- คำถามที่ต้อง join หลาย model
    เช่น "vendor ไหนส่ง bill เกินราคา PO"

- คำถามที่มี filter เฉพาะ
    เช่น "PO เกิน 100,000 บาทที่ยังไม่ได้รับสินค้า"

- คำถามที่ต้อง drill down
    เช่น "SO-2024-0012 ส่งของถึงไหนแล้ว"

- การวิเคราะห์เปรียบเทียบ
    เช่น "เดือนนี้ยอดขายเพิ่มหรือลดจากเดือนที่แล้วเท่าไร"
